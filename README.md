## 📌 Project Overview

Modern cricket generates massive amounts of data from every ball, run, wicket, and player performance. Analyzing such large datasets manually is difficult and time-consuming.

This project focuses on **ICC T20 World Cup 2024 Cricket Analytics**, where raw match data is transformed into meaningful insights using **Data Analytics techniques**.

The project helps in understanding:

- Team Performance 📈  
- Player Statistics 🏏  
- Match Trends 📊  
- Batting & Bowling Analysis ⚡  
- Tournament Insights 🏆

## ❗ Problem Statement

Cricket datasets are:

- Large and complex  
- Mostly unstructured  
- Difficult to analyze manually  
- Hard to visualize clearly  

Important insights like:

- Top player performance  
- Team strengths  
- Match-winning patterns  
- Scoring trends  

often remain hidden.

This project solves this by creating a system that can:

✔ Simplify  
✔ Analyze  
✔ Visualize  
✔ Present data effectively

---

## 🎯 Objectives

- Collect and organize cricket match data  
- Clean and preprocess the dataset  
- Identify KPIs like runs, wickets, strike rate  
- Analyze batting and bowling performance  
- Study match patterns (Powerplay, Middle Overs, Death Overs)  
- Create an interactive dashboard for better understanding

## 🗂 Dataset Description

### Dataset Used:
**ICC T20 World Cup 2024 Match Dataset**

### Type:
**Ball-by-Ball Dataset**

### Dataset Includes:

- Match details (teams, date, venue)  
- Batting data (runs scored, strike rate)  
- Bowling data (wickets, economy rate)  
- Extras (wides, no-balls)  
- Match phases (Powerplay, Middle Overs, Death Overs)

This detailed dataset allows deep performance analysis.

## 💡 Key Insights

### Major Findings:

- Maximum runs are scored during **Death Overs**
- Top-performing players strongly impact match outcomes
- Teams with higher strike rates perform better
- Lower economy rate improves winning chances
- Field-first strategy is commonly preferred
